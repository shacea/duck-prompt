
# DuckPrompt

이 애플리케이션은 PyQt5로 구현된 통합 Prompt Builder입니다.
아래 간단한 안내를 따라 사용할 수 있어요.

---

## 사용 방법

1. `dist\DuckPrompt.exe` 파일을 더블 클릭해서 실행하세요.
2. 실행 후 원하는 프로젝트 폴더를 선택하고, 필요한 파일을 체크한 뒤 프롬프트를 생성해 보세요!
3. 생성된 프롬프트는 바로 복사해서 활용할 수 있어요.

---

## 기능 안내

- **코드 강화 빌더 모드**: 선택한 파일 내용을 합쳐서 LLM에 전달할 프롬프트를 만들어줘요.
- **메타 프롬프트 빌더 모드**: 이미 만들어진 프롬프트를 또 다른 템플릿으로 감싸서 새로운 프롬프트를 생성할 수 있어요.
- **상태 관리**: 작업 상태(체크 파일, 시스템/사용자 입력, 출력 내용 등)을 저장하고 불러올 수 있어요.
- **템플릿 관리**: 자주 쓰는 시스템/사용자 프롬프트를 저장하고 불러오는 기능을 제공해요.
