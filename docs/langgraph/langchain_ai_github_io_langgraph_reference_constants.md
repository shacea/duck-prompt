# Constants

Attributes:

- **`TAG_HIDDEN`**
–



Tag to hide a node/edge from certain tracing/streaming environments.

- **`START`**
–



The first (maybe virtual) node in graph-style Pregel.

- **`END`**
–



The last (maybe virtual) node in graph-style Pregel.
