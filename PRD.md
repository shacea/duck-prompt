
--------------------------------------------  
**[최종 PRD: PyQt5 기반 LLM 프롬프트 생성 도구 - 업데이트 버전]**  
--------------------------------------------

## 1. 개요
- **목적:**  
  - 파일 병합 및 디렉토리 구조 추출 스크립트를 PyQt5 GUI 앱으로 구현  
  - 사용자 친화적 GUI 제공:
    - 프로젝트 폴더 선택 → 디렉토리/파일 구조 필터링 → 특정 파일 및 폴더 선택
    - 선택한 파일들의 내용 + System/User/Developer 프롬프트를 하나의 LLM 프롬프트로 병합
  - 프롬프트 미리보기, 문자수(길이) 실시간 표시, 클립보드 복사 기능 제공
  - 템플릿 관리(YAML) 기능으로 다양한 프로젝트에 재사용 가능
  - 확장자/폴더 제외 필터 설정 가능
  - 디렉토리 구조 표시 및 선택한 폴더의 하위 폴더/파일 전체 구조 반영
  - 프로그램 실행 파일 생성 시 pyinstaller 활용 → 별도 환경 세팅 없이 실행 가능

## 2. 주요 개선사항
1. **폴더 및 하위 구성 자동 선택**  
2. **체크 상태 및 문자수 실시간 반영**  
3. **파일 선택 시 토글 방식**  
4. **선택 파일 영역에서 파일명 클릭 시 체크 토글 기능 추가**  
5. **탐색기 창 우클릭 메뉴로 파일/폴더 삭제, 이름 변경 가능**  
6. **Prompt Output 탭에서 Ctrl+C 시 현재 프롬프트 복사 기능 추가**  
7. **코드 구조 개선**
8. **XML 파일 로드 관련 경로 수정**: `XML_Prompt.md`를 `resources/XML_Prompt.md` 로 이동하여 PyInstaller 빌드 후에도 정상적으로 파일 로드 가능하게 함.

## 3. 주요 기능
- **프로젝트 폴더 선택**
- **디렉토리/파일 구조 표시 및 필터링**
- **파일 및 폴더 체크박스 선택 & 자동 토글**
- **프롬프트 빌드 및 템플릿 관리**
- **프롬프트 생성 및 디렉토리 구조 통합**
- **문자수 실시간 표시 및 클립보드 복사**
- **XML 파싱 및 입력 (파일 생성/수정/삭제)**
- **(업데이트) 선택 파일 영역에서 파일명 클릭 시 체크 토글, 우클릭 메뉴로 파일/폴더 삭제/이름변경, Prompt Output 탭 Ctrl+C 복사, XML 가이드 파일 경로 수정**

## 4. 사용자 경험(UX) 요구사항
- 더 직관적인 상호작용
- Prompt Output 탭에서 단축키(Ctrl+C)로 빠른 복사
- XML 가이드 파일 경로 수정으로 빌드 후에도 정상 동작

## 5. 기술 스택
- Python3, PyQt5, YAML, pyinstaller

## 6. 성능 및 품질 요구사항
- 예외 처리 강화
- 파일 접근 시 에러 처리

## 7. 개발 로드맵(갱신)
- XML_Prompt.md를 resources 폴더로 이동 후 빌드 문제 해결
