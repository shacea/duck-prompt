# PyQt5 기반 LLM 프롬프트 생성 도구
> 파일 병합 및 디렉토리 구조 추출을 위한 통합 GUI 애플리케이션

## 프로젝트 개요

### 목적
- 파일 병합 및 디렉토리 구조 추출 스크립트를 PyQt5 GUI 앱으로 구현
- 사용자 친화적 GUI를 통한 직관적인 워크플로우 제공
- 프롬프트 템플릿과 상태 관리를 위한 통합 시스템 제공

### 핵심 기능
1. **프로젝트 관리**
   - 프로젝트 폴더 선택 및 탐색
   - 디렉토리/파일 구조 필터링
   - 특정 파일 및 폴더 선택

2. **프롬프트 생성**
   - 선택한 파일들의 내용 병합
   - System/User/Developer 프롬프트 통합
   - 실시간 문자수 표시
   - 프롬프트 미리보기
   - 클립보드 복사 기능

3. **리소스 관리**
   - **프롬프트 템플릿**
     - System/User 템플릿 관리
     - 템플릿 로드/저장/삭제/업데이트
   - **상태 관리**
     - 작업 상태 JSON 형식 저장/로드
     - 상태 파일 백업/복구
     - 외부 파일로 Export/Import

## 최근 개선사항

### UI/UX 개선
- 폴더 및 하위 구성 자동 선택 기능
- 체크 상태 및 문자수 실시간 반영
- 파일 선택 시 토글 방식 적용
- 선택 파일 영역에서 파일명 클릭으로 체크 토글
- 탐색기 창 우클릭 메뉴 (파일/폴더 삭제, 이름 변경)
- Prompt Output 탭에서 Ctrl+C로 복사

### 시스템 개선
- Prompts/States 통합 관리 UI
  - ComboBox로 모드 전환
  - 모드별 동적 버튼 라벨 변경
  - States 전체 백업/복구 기능
- 코드 구조 최적화
- XML 파일 로드 경로 개선
- 상태 관리 시스템 강화

## 기술 스택
- Python 3
- PyQt5
- YAML
- pyinstaller

## 파일 구조
```
.
├── app.py              # 메인 애플리케이션
├── main_window.py      # 메인 윈도우 UI (Prompts/States 통합)
├── main_controller.py  # 컨트롤러 (리소스 관리 로직)
├── file_explorer.py    # 파일 탐색기
├── prompt_manager.py   # 프롬프트 관리
├── template_manager.py # 템플릿 관리
├── state_manager.py    # 상태 관리 (백업/복구 포함)
├── config.py          # 설정
├── parse_xml_string.py # XML 파싱
├── utils.py           # 유틸리티
└── resources/
    ├── prompts/       # 프롬프트 템플릿
    │   ├── system/    # 시스템 템플릿
    │   └── user/      # 사용자 템플릿
    └── status/        # 상태 저장 폴더
```
